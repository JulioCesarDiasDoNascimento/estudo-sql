not null -- Não aceita valor nulo

No postgres ao alterar uma coluna não pode ser nula sem o default
Ex: ERRO
    ALTER TABLE pedido_venda.pedido
        ADD COLUMN status varchar(255) not null default;
Ex: CERTO
    ALTER TABLE pedido_venda.pedido
        ADD COLUMN status varchar(255) not null default 'ORÇAMENTO';
